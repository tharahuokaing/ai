## Installation
### Prerequisites
- A modern browser (Chrome, Firefox, or Safari)
- A computer with reasonable specs (RAM 4GB+)
- An internet connection for initial setup

### Steps
1. Download the code from GitHub:
    ```bash
    git clone https://github.com/tharahuokaing/T-Coin.git
    ```
2. Navigate to the project directory:
    ```bash
    cd T-Coin
    ```
3. Open the `index.html` file in your browser.
